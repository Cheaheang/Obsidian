**[Asp.NET](http://Asp.NET) Core** is a cross-platform, high-performance, open-source framework for building modern, cloud-enabled web applications and services

[asp.net](http://asp.net) core app can be hosted on window, window linux and mac

[asp.net](http://asp.net) have multiple type:

- MVC
- Web API
- Razor Pages
- Blazor

[Asp.Net](http://Asp.Net) Web Forms: Performance issues due to server events and view-state

[Asp.Net](http://asp.net/) MVC : Performance issue due to build on [asp.net](http://asp.net) web form and lack support of cross plateform. Also slightly cloud-friendly

[Asp.Net](http://Asp.Net) Core: Performance faster and cross-plateform. Cloud friendly
